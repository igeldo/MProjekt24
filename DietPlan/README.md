# DietPlan

DietPlan is an application that enables the user to calculate and track their intake as well as their total metabolism.

## Overview of used formulas and equations
Total metabolism includes activity and supply. If the result is positive, the user may gain some weight and if the
result is negative the user may lose some weight.

Basal metabolism is calculated using three different valid methods. These are the Harris-Benedict formula, the Mifflin-St. Jeor equation and Schofield equation.
The formulas can be found here:
- Harris-Benedict: https://www.omnicalculator.com/health/bmr-harris-benedict-equation#what-is-the-harris-benedict-equation
- Mifflin-St. Jeor: https://www.omnicalculator.com/health/bmr
- Schofield: https://en.wikipedia.org/wiki/Schofield_equation

Activity is calculated as a factor based on the amount of sport the user does per week. The used factors can be found here:
- https://www.omnicalculator.com/health/tdee

Supply is provided by the user entering the amount of grams eaten for every food and kcal per 100g. The calories of foods can be found here:
- https://www.calories.info/

## Explanation of program execution
Specify the number of individuals you wish to enter into the application. Refer to the main section for any modifications.

You enter your name, date of birth, weight, height and gender as well as the food, the amount of grams eaten and the calories per 100g.

## Disclaimer
This program does not serve as a guideline for losing or gaining weight, but is only intended to give an overview of the total metabolism.
